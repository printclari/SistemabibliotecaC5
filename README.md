## Descripción del proyecto

Este proyecto consiste en el desarrollo de un sistema de gestión de biblioteca realizado en **Python** y ejecutado por consola.

El objetivo del sistema es administrar el préstamo y la devolución de libros, permitiendo registrar libros y usuarios, controlar la disponibilidad de ejemplares, calcular multas por demora y consultar estadísticas sobre la actividad de la biblioteca.

Durante el desarrollo se aplicaron los contenidos vistos en la asignatura, como estructuras condicionales y repetitivas, funciones, programación modular, validaciones, manejo básico de errores y programación orientada a objetos.

---

## Funcionalidades

El sistema permite:

* Registrar libros.
* Registrar usuarios.
* Realizar préstamos de libros.
* Registrar devoluciones.
* Calcular multas por demora.
* Buscar libros por título.
* Buscar usuarios por nombre o DNI.
* Mostrar el listado de libros.
* Mostrar préstamos activos.
* Mostrar préstamos de un usuario.
* Eliminar libros (cuando no poseen préstamos activos).
* Eliminar usuarios (cuando no poseen préstamos activos).
* Mostrar el ranking de libros más solicitados.
* Mostrar la cantidad total de préstamos realizados.
